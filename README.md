# Vue.js Todo App

A simple and elegant todo list application built with Vue.js 3.

## Features

- Add new todos
- Mark todos as completed
- Delete todos
- Filter todos by status (All, Active, Completed)
- Clear all completed todos
- Persistent storage using localStorage
- Beautiful gradient UI with smooth animations

## How to Use

1. Simply open `index.html` in your web browser
2. Type a new todo in the input field and press Enter or click "Add"
3. Click the checkbox to mark a todo as completed
4. Click "Delete" to remove a todo
5. Use the filter buttons to view all, active, or completed todos
6. Click "Clear Completed" to remove all completed todos at once

## Technologies Used

- Vue.js 3 (via CDN)
- HTML5
- CSS3
- JavaScript (ES6+)
- localStorage for data persistence

## Features Explained

- **Reactive Data Binding**: Todo state is automatically synchronized with the UI
- **Computed Properties**: Filtered lists are efficiently calculated
- **Local Storage**: Your todos persist even after closing the browser
- **Responsive Design**: Works great on desktop and mobile devices

## No Build Process Required

This is a simple Vue.js application that uses the CDN version, so no build process or npm installation is required. Just open the HTML file in your browser and start using it!
